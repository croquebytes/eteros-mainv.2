# References Folder (Generic)

This folder is for **supporting material** that LLMs and humans can refer to while working
on **`PROJECT_NAME`**.

Suggested contents:

- High-level design docs and specs.
- Architecture and systems documents.
- UI/UX wireframes or notes.
- Lifecycle / monetization / campaign specs (if relevant).
- LLM session notes and logs.
